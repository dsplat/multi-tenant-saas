# HTTP 端点总览

**最后更新**: 2026-06-29

---

## 约定

- **Base URL**: `https://<api-domain>/api/v1`
- **鉴权**: 除标注「无需认证」外，均需 `Authorization: Bearer <token>`（Sanctum）
- **限流**: 认证后接口默认 `throttle:api`（60/min，按用户 ID）
- **响应**: 统一 `{ "success": bool, "data": ..., "message": "..." }`
- **权限**: `rbac.permission:<node>` 表示需要对应 RBAC 权限
