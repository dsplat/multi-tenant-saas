# 主题切换和 UI 框架配置

框架支持多种 UI 框架和主题切换，方便开发者选择最适合的技术栈。

---

## 支持的 UI 框架

| 框架 | 说明 | 特点 |
|------|------|------|
| **Element Plus** | 饿了么开源组件库 | 组件丰富、文档完善、中文友好 |
| **Ant Design Vue** | 蚂蚁金服组件库 | 设计规范、企业级、国际化 |
| **Naive UI** | Vue 3 原生组件库 | 轻量、TypeScript 友好、性能优秀 |

---

## 快速开始

### 安装依赖

```bash
cd resources/js/admin
npm install
```

### 选择 UI 框架

#### Element Plus（默认）

```bash
npm install element-plus @element-plus/icons-vue
npm run dev:element-plus
```

#### Ant Design Vue

```bash
npm install ant-design-vue
npm run dev:ant-design
```

#### Naive UI

```bash
npm install naive-ui
npm run dev:naive-ui
```

## 主题配置

### 主题模式

支持三种主题模式：

- **浅色模式** (light) - 默认
- **深色模式** (dark) - 护眼
- **自动模式** (auto) - 跟随系统

```typescript
import { useTheme } from '@multi-tenant-saas/ui-core'

const { setThemeMode } = useTheme()

// 切换到深色模式
setThemeMode('dark')

// 跟随系统
setThemeMode('auto')
```

### 主色调

支持自定义主色调：

```typescript
import { useTheme } from '@multi-tenant-saas/ui-core'

const { setPrimaryColor } = useTheme()

// 设置主色调
setPrimaryColor('#67c23a') // 绿色
```

预设颜色：
- `#409eff` - 默认蓝
- `#67c23a` - 绿色
- `#e6a23c` - 橙色
- `#f56c6c` - 红色
- `#909399` - 灰色
- `#00d1b2` - 青色
- `#b388ff` - 紫色
- `#ff6b6b` - 粉红

### 圆角大小

支持自定义圆角：

```typescript
import { useTheme } from '@multi-tenant-saas/ui-core'

const { setBorderRadius } = useTheme()

// 设置圆角（0-12px）
setBorderRadius(8)
```

## 使用主题组件

### ThemeSwitcher

主题切换按钮：

```vue
<template>
  <ThemeSwitcher />
</template>

<script setup>
import { ThemeSwitcher } from '@multi-tenant-saas/ui-core'
</script>
```

### ColorPicker

颜色选择器：

```vue
<template>
  <ColorPicker />
</template>

<script setup>
import { ColorPicker } from '@multi-tenant-saas/ui-core'
</script>
```

### ThemeSettings

主题设置面板：

```vue
<template>
  <el-button @click="showSettings = true">主题设置</el-button>
  <ThemeSettings v-model:visible="showSettings" />
</template>

<script setup>
import { ref } from 'vue'
import { ThemeSettings } from '@multi-tenant-saas/ui-core'

const showSettings = ref(false)
</script>
```

## 自定义主题

### CSS 变量

框架使用 CSS 变量实现主题切换，可以自定义：

```css
/* 自定义变量 */
:root {
  --primary-color: #409eff;
  --border-radius: 4px;
  --bg-color: #ffffff;
  --text-color: #303133;
}

/* 深色主题 */
.dark {
  --bg-color: #141414;
  --text-color: #e5eaf3;
}
```

### 自定义 UI 框架

创建自定义适配器：

```typescript
import { registerAdapter } from '@multi-tenant-saas/ui-core'

const customAdapter = {
  name: 'custom',
  install(app) {
    // 安装自定义组件库
  },
  getComponents() {
    return {
      Button: 'custom-button',
      Input: 'custom-input',
      // ...
    }
  },
}

registerAdapter(customAdapter)
```

## 最佳实践

1. **选择合适的 UI 框架**
   - 国内项目推荐 Element Plus（中文文档完善）
   - 国际化项目推荐 Ant Design Vue（设计规范）
   - 追求性能推荐 Naive UI（轻量级）

2. **主题一致性**
   - 使用 CSS 变量保持主题一致性
   - 避免硬编码颜色值

3. **按需导入**
   - 使用 Tree Shaking 减少打包体积
   - 只导入需要的组件

4. **深色模式适配**
   - 测试所有页面在深色模式下的显示
   - 确保文字对比度符合无障碍标准
